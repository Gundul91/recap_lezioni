      Parametri di default:
      function greet(name = 'Student', greeting = 'Welcome') {
        return `${greeting} ${name}!`;
      }
      I valori inseriti nelle tonde sono i valori di default nel caso non vengano passati quando viene chiamata la funzione

      greet(); // Welcome Student!
      greet('James'); // Welcome James!
      greet('Richard', 'Howdy'); // Howdy Richard!

      Parametri di default con array:
      function createGrid([width = 5, height = 5] = []) {
        return `Generates a ${width} x ${height} grid`;
      }
      il "= []" serve come array vuota di default in caso venga chiamata la funzione senza parametri

      Parametri di default con oggetti:
      function houseDescriptor({houseColor = 'green', shutterColors = ['red']} = {}) {
        return `I have a ${houseColor} house with ${shutterColors.join(' and ')} shutters`;
      }
      il "= {}" serve come oggetto vuota di default in caso venga chiamata la funzione senza parametri
