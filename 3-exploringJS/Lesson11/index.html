Proxie:
      const richard = {status: 'looking for work'};
      const handler = {
          get(target, propName) {
              console.log(target); // the `richard` object, not `handler` and not `agent`
              console.log(propName); // the name of the property the proxy (`agent` in this case) is checking
          }
      };
      const agent = new Proxy(richard, handler);
      agent.status; // logs out the richard object (not the agent object!) and the name of the property being accessed (`status`)
      Il proxie permette di intercettare una chiamata all'oggetto "target" ed eseguirci altro codice
      ci sono 13 diverse "trappole" in questo caso è usato il "get"

Restituire un valore dalla funzione:

      function* getEmployee() {l'asterisco dopo "function" indica che è un generatore
          console.log('the function has started');

          const names = ['Amanda', 'Diego', 'Farrin', 'James', 'Kagure', 'Kavita', 'Orit', 'Richard'];

          for (const name of names) {
              yield name;yeld indica dove deve fermarsi il codice e quello dopo yeld viene restituito
          }

          console.log('the function has ended');
      }

      const generatorIterator = getEmployee();
      let result = generatorIterator.next();
      result.value // is "Amanda"
      result.value perchè result è un iteratore object(value, done)

      generatorIterator.next().value // is "Diego"
      generatorIterator.next().value // is "Farrin"
      il "= {}" serve come oggetto vuota di default in caso venga chiamata la funzione senza parametri
